The dataset for the challenge will be released on February 15th, 2020. The proposed dataset will consist of sixty high quality, Full HD video sequences (both RGB and Thermal Infrared), spanning multiple occurrences of multi-scale UAVs. We will build a challenge server on CodaLab which will be open for submission soon afterwards.

4. Division Rules

Test set: Select all data files involved in "false object", do not participate in training, mainly for verification.

Validation set: According to the ratio of 3:7 randomly divided the remaining data, classification on the scene of "big", "middle", "small", "day" and "night", right in the file data according to the number of different distribution division (for example, big UAVs-day have 50, big UAVs-night are 20, if needs in extraction of 20 UAVs, day:night = 14:6), as far as possible to disrupt the same video data sequence.

Train set: Data other than validation set.

The final partition ratio is: Train set: Validation set: Test set =5:2:3.
